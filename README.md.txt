Todo App
Overview
This Todo App is a simple web-based application that allows users to manage their daily tasks efficiently. It enables users to add, edit, delete, and persist their todos in the browser's local storage. The design is responsive and user-friendly, utilizing modern HTML, CSS, and JavaScript features.

Features
Add Todos

Users can add new tasks using an input field and a "Add" button.
View Todos

All added tasks are displayed in a clean, card-style layout.
Edit Todos

Users can click the "Edit" button to modify an existing task.
Delete Todos

Users can remove tasks using the "Delete" button.
Persist Todos

All tasks are stored in the browser's localStorage, ensuring persistence even after refreshing the page.
How to Use
Open the Application

Open the HTML file in any modern browser.
Add a Task

Enter a task in the input field at the top and click the "Add" button.
Edit a Task

Click the "Edit" button (pencil icon) next to a task. Modify the task in the input field, then click "Add" again to save the changes.
Delete a Task

Click the "Delete" button (trash icon) next to a task to remove it.
View Tasks

All tasks are displayed below the input field and will persist across browser sessions.
Technologies Used
HTML5: For the structure of the app.
CSS3: For styling the app, including responsiveness and layout.
JavaScript: For app functionality, including adding, editing, deleting, and persisting tasks.
Font Awesome: For using intuitive icons (Edit and Delete buttons).
Google Fonts: For modern typography with the Inter and Press Start 2P fonts.
Code Breakdown
HTML

Provides the structure of the app, including an input field, buttons, and a main container for tasks.
CSS

Creates a modern, responsive design using flexbox and custom styles.
JavaScript

Implements the core functionality:
Adding todos to the list.
Editing or deleting tasks.
Persisting the list using localStorage.
Local Storage
The app uses the browser's localStorage to save todos as JSON objects. This ensures that tasks remain available even if the page is refreshed or closed.

Getting Started
Clone or download the repository.
Open the file index.html in a browser.
Start managing your tasks!
